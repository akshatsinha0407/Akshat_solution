Exercise 19.1 – Bootstrap 5 + Sass Setup (npm)

Follow these steps to set up a Bootstrap 5 project with Sass customization.

mkdir my-bootstrap-project
cd my-bootstrap-project
npm init -y

npm install bootstrap sass

This installs Bootstrap source files and the Sass compiler.

Create scss/custom.scss:

// 1. Override Bootstrap variables BEFORE importing Bootstrap
$primary: #e74c3c;
$border-radius: 0.75rem;
$font-family-base: 'Roboto', sans-serif;

// 2. Import Bootstrap source
@import "../node_modules/bootstrap/scss/bootstrap";

# Compile once
npx sass scss/custom.scss css/custom.css

# Watch for changes
npx sass --watch scss/custom.scss css/custom.css

Then link css/custom.css in your HTML instead of the CDN Bootstrap CSS.

✅ After setup, all Bootstrap components will use your custom colors and settings!
