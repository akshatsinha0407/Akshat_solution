Bootstrap 5 is the latest version of the world's most popular front-end open source toolkit. It includes HTML, CSS, and JavaScript-based components for rapidly building responsive websites and web applications.

Bootstrap 5 drops jQuery as a dependency, adds a new grid tier (xxl), includes new utilities API, improved forms, new components like offcanvas and floating labels, and uses CSS custom properties.

Yes! Bootstrap is completely free and open-source, licensed under MIT. You can use it in personal and commercial projects without any cost.

You can install Bootstrap 5 via CDN link (easiest), npm (npm install bootstrap), or download the compiled files from getbootstrap.com. Link the CSS in your <head> and JS bundle before </body>.
