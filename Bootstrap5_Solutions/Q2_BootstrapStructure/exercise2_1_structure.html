Exercise 2.1 – Bootstrap Directory Structure

When you download or install Bootstrap via npm, you get the following folder structure:

📁 bootstrap/
📁 css/
  • bootstrap.css – Full unminified CSS
  • bootstrap.min.css – Minified CSS for production
  • bootstrap.rtl.css – Right-to-left language support
  • bootstrap-grid.css – Grid system only
  • bootstrap-utilities.css – Utility classes only
  • bootstrap-reboot.css – CSS reset/normalize
📁 js/
  • bootstrap.js – Full JS (requires Popper.js separately)
  • bootstrap.min.js – Minified JS
  • bootstrap.bundle.js – JS bundled WITH Popper.js
  • bootstrap.bundle.min.js – Minified bundle (recommended)
  • bootstrap.esm.js – ES Module format
📁 icons/ (Bootstrap Icons – separate package)
  • Contains 1800+ SVG icon files
  • Use via <i class="bi bi-heart"></i> after linking bootstrap-icons.css
